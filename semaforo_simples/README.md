    1. O LED verde (pino 11) fica aceso por 3 segundos.
    2. Depois apaga, e o LED amarelo (pino 12) pisca 3 vezes.
    3. Depois apaga, e o LED vermelho (pino 13) fica aceso por 3 segundos.
    4. O ciclo recomeça.
